Scratch Game Creation

    Low Budget CS Soviet

Description


  • Context: Counter Strike and Duck Hunt
  • Scoring System: Each chicken gave different points
  • Randomness System: Asks test your might; button used for that purpose
  • Difficulty: Easy and Insane
  • Game End: 3 ending scenarios determined by the dangerous looking debris

Reflection

I made this game solely for entertainment and my inspiration came from a generic game of
duck hunt with a little spice of counterstrike theme.

If I had more resources I would the aspect of the game by adding mechanics to make the game
more enjoyable. For an example, one mechanic I would add is to the aiming mechanic used in FPS.

Success

  1. Audio played when events occur
  2. Game ends and restarts

Obstacles

  1. Getting audio to synchronize with events
  2. Coming up with the sprites and their tasks

Algorithm

An algorithm that was included with the game was the mouse detection. This algorithm is important
because it helps gives the player a chance to play the game and keep track of their score.
