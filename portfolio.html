Python Story Mode

Challenged

In this python program, it introduces a short story between two friends who will be clashing.
It aims to show readers that although it is a match, friends are meant to stick around for you instead
of hurting them.

Python Story Reflection

In order, to create a perfectly running program which depicts the script, we used strings and functions. Although it may seem easy, there were some difficulties
when starting this program. One of which was deciding which scenes to add in and have the readers input their choices.
To resolve it, we took it a step at a time and used our current knowledge to give readers choices to alter the story.
Another issue we faced was
the story itself. We wanted something simple and out of the ordinary because we deemed it entertaining to.
In order to accomplish that we brain stormed some ideas, and we came up with
diverse ideas. These ideas were something out of our experience and one of which was games that we have played.
We were going to try to put the reader into an intense situation in a game that we have played and let them decide for themselves.
We though about it, but we also knew that no one was going to have much interest in the game
we played so we came up with airsoft match. At first, the script came out very long and inflexible so we decided to shorten it and
make it where the reader basically play god. In the end, we concluded the plot to be a scrirmish
between two friends because it was short and interesting.

Reflection

I made this game solely for entertainment and my inspiration came from a generic game of
duck hunt with a little spice of counterstrike theme.

Additionally, If I had more resources I would improve the aspect of the game by adding mechanics to make the game
more enjoyable. For an example, one mechanic I would add is to the aiming mechanic used in FPS.
